# Privacy Policy for Py3Code IDE Pro

**Last updated: January 19, 2026**

## Introduction

Welcome to Py3Code IDE Pro! This Privacy Policy explains how we collect, use, and protect your information when you use our Python IDE mobile application ("App").

## Information We Collect

### Information Stored Locally

Py3Code IDE Pro stores the following data locally on your device:

- **Python Files**: Your Python source code files and projects
- **App Settings**: Theme preferences, font size, editor configuration
- **Installed Packages**: List of Python packages you install from PyPI
- **Execution History**: Recent code execution results

### Information Collected Automatically

When you use Py3Code IDE Pro, we may automatically collect:

- **Device Information**: Device type, operating system version
- **Usage Data**: App sessions and features used
- **Crash Reports**: Technical information when the app crashes

### Third-Party Services

Our App uses the following third-party services that may collect information:

#### Google AdMob
- Used to display banner advertisements
- May collect device identifiers and usage data for personalized ads
- Subject to [Google's Privacy Policy](https://policies.google.com/privacy)

#### PyPI (Python Package Index)
- Used when you install Python packages
- Package download requests are sent to PyPI servers
- Subject to [PyPI's Terms of Use](https://pypi.org/policy/terms-of-use/)

## How We Use Your Information

We use the collected information to:

- Store and manage your Python files locally
- Save your app preferences and settings
- Display relevant advertisements
- Improve app performance and user experience
- Debug and fix issues

## Data Storage

- **All User Data is Local**: Your Python files, settings, and execution history are stored only on your device
- **No Cloud Sync**: We do not upload or sync your code to any server
- **No Account Required**: The app does not require account creation

## Internet Usage

Py3Code IDE Pro requires internet access for:

- Displaying advertisements (AdMob)
- Installing Python packages from PyPI
- Initial loading of Pyodide engine (WebAssembly, if used)

**Note**: The core Python execution via Chaquopy engine works fully offline with bundled packages.

## Children's Privacy

Py3Code IDE Pro is suitable for users of all ages learning Python programming. We do not knowingly collect personal information from children under 13. The app does not require account creation or personal data input.

## Advertising

We use Google AdMob to display banner ads in the app. AdMob may use cookies and similar technologies to serve ads based on your prior visits to our app or other apps. You can opt out of personalized advertising by visiting [Google's Ad Settings](https://www.google.com/settings/ads).

## Your Rights

You can:

- **Delete All Data**: Uninstall the app to remove all locally stored data (files, settings, packages)
- **Export Your Files**: Copy your Python files before uninstalling
- **Opt-out of Personalized Ads**: Adjust your device's ad preferences in Settings > Google > Ads

## Data Security

- All your code and files remain on your device
- We do not have access to your Python files or code
- We implement appropriate security measures to protect the app

## Permissions

The app requests the following permissions:

- **Internet**: Required for ads and package installation
- **Network State**: Check connectivity status
- **Storage** (Android 12 and below): Access files for import/export

## Open Source Libraries

Py3Code IDE Pro uses the following open-source components:

- **Chaquopy**: Python runtime for Android
- **Pyodide**: Python WebAssembly runtime (optional)
- **NumPy, Pandas, Matplotlib**: Bundled Python packages
